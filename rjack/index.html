RJack: Java component gems for JRuby

Project | Download

Summary

RJack provides common and broadly used open source java components as gems for use with JRuby. The gems additionally provide facades or adapters where needed to support use and setup of these java components from ruby code. RJack helps bring the advantages of RubyGems to existing Java applications, and lowers barriers for these applications to take advantage of better ruby facilities in such areas as file system and command line interaction, Ruby or DSLs as better "configuration", etc.

In general, rjack gems will closely track upstream java project releases (as opposed to individual project jars). Gem versions will be based on the java project version plus an additional gem release number.

Gems

Logging

  • slf4j gem
  • logback gem

Gems for SLF4J and Logback provide a complete toolkit for unifying and controlling logging in a heterogeneous application containing ruby and java components. The slf4j gem may simply be used to load SLF4J jars that a java component is dependent on, but also offers a ruby core Logger compatible logging adapter. The logback gem provides a facade for programmatic logback configuration in ruby.

HTTP Server

  • jetty gem
  • jetty-jsp gem

Gem packaging of the Jetty Web Server. The jetty gem includes an extensible ServerFactory for Server setup in ruby. The jetty-jsp companion gem adds JSP support when needed.
